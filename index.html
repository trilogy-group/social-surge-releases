Date: 30 Aug, 2024

Release 4.6.2

Date: 28 Aug, 2024

  • The Script stage in the Video Builder is now built in parts based on the script outline to prevent token limit overflow. This allows building much longer scripts.
  • 'Regenerate with Feedback' and 'Next Stage' api calls in the Video Builder are now made async to prevent timeouts; while the UI polls for status.
  • The default output token limit for LLM calls has been increased to 8k from 4k.

Release 4.5.0

Date: 20 Aug, 2024

  • Video Builder
    • Concept Initiation: Starts from creating the concept, setting a solid foundation for your video project.
    • Create with Ease: Introducing our new Video Builder, a tool designed to help you generate and visualize new ideas for your video projects. It guides you through each stage of the creation process.
    • Guided Workflow: With step-by-step assistance, you'll find it easier than ever to bring your ideas to life.
    • Dynamic Feedback Integration: The Video Builder can take feedback at any stage and update accordingly, allowing for a flexible and iterative creation process.
    • Navigational Flexibility: Easily navigate back and forth between all changes and stages, ensuring you can refine and adjust your video content as needed.
  • Quality Control for Second Brains
    • The QC prompt can be edited from the new 'QC Prompts' page under Admin section.
    • QC results are automatically generated when either a Second Brain or the QC Prompt is edited.
    • Detailed QC results are displayed on clicking the 'Edit' button for a Second Brain.
    • The QC prompt has been refined to detect Markdown structure first and then subsequent evaluation rules.
    • 'Download' button has been removed the Second Brains page to keep the UI simple.

Release 4.1.0

Date: 8 Aug, 2024

  • Reports Simplification
    • Reports now support interactive widgets that can be generated using the conversational interface. You can now add "Report React Component", to a section in addition addomh text sections.
    • The "Reports" menu item has now been removed. Reports can be accessed via the channels details page.
    • Reports are now auto-generated on video ingestion.
  • UI/UX Fixes - This release covers a number of UI/UX fixes to the add channel, channel analysis history, and channel details pages
    • The 'Clear Search', 'Clear Date' buttons are updated to be different from 'Fetch Videos' button and to be less prominent.
    • The highlight color when selecting video thumbnail has been changed from black to blue.
    • The 'Edit Prompts' button is removed from the 'Add Channel' page.
    • The 'Begin Video Analysis' button is changed to 'Begin Analysing X selected videos' to dynamically show number of selected videos.
    • Stop Date column is removed on Channel Analysis History page.
    • Status and Results are separated into separate columns and display corresponding information.
    • Error messages have been improved and will reflect for new executions. Old stuck executions have been handled.
    • All videos including ones that have failed will be shown on clicking 'Details' button on the Channel Analysis History page.
    • UX for View/Download buttons has been improved on the Details page.
